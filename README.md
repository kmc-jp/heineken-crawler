# heineken-crawler

Hayai Kensaku Pukiwiki Crawler & Client

## Requirements

Python 3.x

## Usage

- Setup

```shell
$ cp config.py.example config.py
$ edit config.py
```

- Crawl

```shell
$ python3 crawl.py
```

- Clients for dev

```shell
# show help
$ python3 dev-client.py -h
```
